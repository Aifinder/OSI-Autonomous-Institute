# Release Roadmap

## Release 0.1 — Governed Orchestrator Foundation

Outcome: a working local control plane that can accept structured goals, create work packages, manage dependencies, route reviews, decide routine approvals, record audits, and produce founder escalations.

Required capabilities:

- typed domain models and schemas
- persistent work queue
- dependency graph
- orchestrator state machine
- agent registry and task router
- policy and budget engine
- review-gate framework
- approval engine
- artifact registry
- append-only audit log
- escalation packet generator
- CLI administration
- automated tests

## Release 0.2 — Autonomous Knowledge Pipeline

Outcome: the orchestrator can generate, review, revise, approve, and register institutional documents with minimal human involvement.

Required agents:

- Roadmap Agent
- Planning Agent
- Document Agent
- Architecture Reviewer
- Constitutional Reviewer
- Evidence Reviewer
- Duplication and Contradiction Reviewer
- Terminology Validator
- Quality Reviewer
- Revision Agent

## Release 0.3 — Persistent Operations

Outcome: reliable continuous execution across restarts and controlled parallel workers.

Capabilities:

- checkpointing and recovery
- concurrency controls
- budgets and rate limits
- scheduled runs
- dashboard
- escalation inbox
- daily exception report
- operational metrics

## Release 0.4 — Knowledge Intelligence

Outcome: the system understands institutional coverage, dependencies, contradictions, provenance, and downstream impact.

Capabilities:

- knowledge graph
- semantic retrieval
- contradiction detection
- coverage analysis
- citation lineage
- change-impact analysis
- automatic roadmap reprioritization

## Release 1.0 — Institutional Factory

Outcome: a reusable platform capable of engineering OSI and additional ventures from governed specifications.

Capabilities:

- research pipelines
- operating-system generation
- books and courses
- certifications
- software product pipelines
- release management
- institutional digital twins
- multi-venture portfolio orchestration
